HP iLO C7000 Credential Reset

Use a USB-based script to forcibly reset Onboard Administrator credentials

Dec 27, 2021

As previously mentioned in ESXi 6.0 to 6.5 Upgrade, I recently took over a manufacturing site where some administrative passwords that weren’t known. In particular, there were two HP Blade C7000 chassis with no credentials for the Blade Onboard Administrator.

The easiest way to reset the Administrator credential is by plugging a USB key (with a reset script) in the chassis, then use the Insight LCD to run the script.

Create a new text file containing only the following string: SET USER PASSWORD "Administrator" "password" and save it with a .cfg extension on a USB key. On the Insight LCD, select USB Menu > Restore Configuration > usb://d1/resetadmin.cfg

This will reset the Administrator password to password. Launch your browser of choice, connect to the BOA and log in with Administrator: password
